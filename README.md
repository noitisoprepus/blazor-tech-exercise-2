# blazor-tech-exercise-2

## Prerequisites
- .NET 8.0 SDK
- Visual Studio 2022 (or later)

## Build & Run

### Using Command Line

**Build:**
```bash
dotnet build
```

**Run:**
```bash
dotnet run
```
The application will be available at http://localhost:5198

### Using Visual Studio

1. Open the solution file `BlazorTestProject.sln` in Visual Studio
2. Press **Ctrl+Shift+B** or go to **Build > Build Solution**
3. Press **F5** or click the **Run** button to start the application
4. The application will open in your default browser at http://localhost:5198
